# CDA LISTA - Guia Comercial de Conceição do Araguaia

![Versão](https://img.shields.io/badge/vers%C3%A3o-1.0.0-blue)
![Tecnologias](https://img.shields.io/badge/tech-HTML5%20%7C%20CSS3%20%7C%20JS-orange)

O **CDA LISTA** é um guia comercial digital robusto e responsivo, projetado para conectar os cidadãos de Conceição do Araguaia (PA) ao comércio e prestadores de serviço locais de forma rápida e intuitiva.

---

## Funcionalidades Principais

- **Busca Inteligente:** Filtro em tempo real por nome da empresa ou categoria.
- **Grid Responsivo:** Visualização otimizada com **6 colunas no PC** e 2 colunas no Mobile.
- **Modal de Detalhes:** Janela dedicada para cada empresa com endereço, descrição e link direto para WhatsApp.
- **Ordem Alfabética Automática:** Lógica em JavaScript que organiza todos os cadastros de A a Z.
- **Categorias Dinâmicas:** Navegação por abas para filtrar setores específicos (Saúde, Gastronomia, Lazer, etc).
- **Popup de Conversão:** Banner promocional com contador regressivo de 30 segundos para anúncios premium.
- **Telefones de Emergência:** Seção fixa com números úteis da cidade (PM, SAMU, Bombeiros).

## Tecnologias Utilizadas

Este projeto foi construído com foco em performance e SEO, utilizando apenas tecnologias nativas (Vanilla):

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Layout moderno utilizando **Flexbox** e **CSS Grid** (6 colunas).
* **JavaScript (ES6+):** Manipulação de DOM, lógica de filtragem e persistência de dados simples.
