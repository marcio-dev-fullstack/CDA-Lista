# CDA LISTA - Guia Comercial de Conceição do Araguaia

![Versão](https://img.shields.io/badge/vers%C3%A3o-1.0.0-blue)
![Tecnologias](https://img.shields.io/badge/tech-HTML5%20%7C%20CSS3%20%7C%20JS-orange)

O **CDA LISTA** é um guia comercial digital robusto e responsivo, projetado para conectar os cidadãos de Conceição do Araguaia (PA) ao comércio e prestadores de serviço locais de forma rápida e intuitiva.
